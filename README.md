# XMLDiffStudio

Aplicacion de escritorio para comparar dos archivos `XML` o `JSON` y detectar diferencias de estructura y valores.

## Funciones

- Compara `XML` contra `XML`
- Compara `JSON` contra `JSON`
- Detecta cambios, nodos agregados y nodos eliminados
- Muestra la ruta exacta de cada diferencia
- Exporta un reporte a `TXT`

## Uso

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python .\XMLDiffStudio.py
```
